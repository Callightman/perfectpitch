# Pitchy — Legal & Support pages

Static site hosted via GitHub Pages for the **Pitchy** iOS app. Provides the
Privacy Policy, Support/Contact, Terms of Use, and a landing page required for
App Store distribution.

- Privacy Policy — `privacy.html`
- Support / Contact — `support.html`
- Terms of Use (EULA) — `terms.html`
- Landing — `index.html`

The app's source code lives in a separate, private location and is intentionally
not part of this repository.
